Architecture Decision (Approved)

The Premium AI Receptionist uses OpenAI Realtime API, which requires a persistent WebSocket connection.
n8n does not host or proxy live audio.

Therefore:

Live voice conversation happens externally:

Twilio Media Streams ↔ External WebSocket Server ↔ OpenAI Realtime API

n8n handles orchestration only:

Call start

Call end

Data ingestion

CRM logging

Intake handoff

This is intentional and correct.
