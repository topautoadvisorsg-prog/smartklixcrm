What exact manual steps are required in n8n?

What endpoints should exist once setup is complete?
For each Webhook Trigger node, an endpoint like /webhook/<path> (e.g., /webhook/google/gmail) will be available and respond to POST requests if the workflow is active.

2. Google Workspace Integrations in n8n
What workflows should exist?
For each Google Workspace integration (Gmail, Sheets, Calendar, etc.), there should be a workflow with:

A Webhook Trigger node (for inbound events).
The relevant Google node (e.g., Gmail, Google Sheets).
Any processing or logic nodes required.
What webhook paths should be active?
The path is defined in the Webhook Trigger node (e.g., /webhook/google/gmail). Each active workflow with a webhook node exposes its path as an endpoint.
