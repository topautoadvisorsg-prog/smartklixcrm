⚙️ BACKEND AUDIT
CRITICAL GAPS
1. No Request Validation Layer
Every endpoint trusts input blindly:

// BAD:
app.post("/api/ai/chat/internal", async (req, res) => {
  const validated = schema.parse(req.body); // Can throw
  ...
});
// NEEDED:
app.post("/api/ai/chat/internal", validateRequest(schema), async (req, res) => {
  // Guaranteed valid here
});

2. No Rate Limiting
// ANY tool can be called unlimited times
await executeAITool("send_email", {...}); // Could send 10,000 emails

NEEDED:

import rateLimit from 'express-rate-limit';
const aiChatLimiter = rateLimit({
  windowMs: 60 * 1000, // 1 minute
  max: 10, // 10 requests per minute
  message: 'Too many AI requests, please try again later'
});
app.post("/api/ai/chat/internal", aiChatLimiter, ...);

3. No Background Job Queue
All AI operations are synchronous:

// Blocks the request thread
const result = await architect.chat(message); // Could take 30+ seconds

NEEDED:

// Queue long-running tasks
const jobId = await jobQueue.add('ai-chat', { message, userId });
res.json({ jobId, status: 'processing' });
// Poll for results
GET /api/ai/jobs/{jobId} → { status: 'completed', result: {...} }

4. No Webhook Signature Verification
N8N webhook accepts ANY request:

app.post("/api/events/update", async (req, res) => {
  // ANYONE can post here - no signature check
});
